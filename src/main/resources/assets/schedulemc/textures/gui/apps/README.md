# Smartphone App Icons

Dieses Verzeichnis enthält die Icons für die Smartphone-Apps im Spiel.

## Platzhalter-Icons

Die aktuellen Icons sind einfarbige Platzhalter. Sie können diese durch eigene Icons ersetzen.

## Icon-Spezifikationen

- **Größe**: 48x48 Pixel
- **Format**: PNG
- **Transparenz**: Optional (Alpha-Kanal wird unterstützt)

## Icons austauschen

1. Erstellen Sie ein 48x48 PNG-Bild
2. Benennen Sie es entsprechend (z.B. `app_map.png`)
3. Ersetzen Sie die Datei in diesem Verzeichnis
4. Starten Sie das Spiel neu oder laden Sie die Ressourcen neu (F3+T)

## Design-Tipps

- Verwenden Sie klare, einfache Symbole
- Achten Sie auf gute Kontraste
- Testen Sie die Icons im Spiel
- Überlegen Sie sich ein einheitliches Design für alle Apps
